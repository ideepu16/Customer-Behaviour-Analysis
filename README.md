# Customer-Behaviour-Analysis
This project focuses on analyzing customer purchasing behavior from an e-commerce dataset to extract meaningful insights. It includes data preprocessing, feature engineering, customer segmentation using RFM (Recency, Frequency, Monetary) analysis, and clustering with K-Means.

# 🛒 Customer Behaviour Analysis

## 📌 Overview
This project analyzes customer behavior in an e-commerce dataset using Python and Machine Learning techniques. It focuses on identifying customer segments and purchasing patterns to help businesses make data-driven decisions.

---

## 📂 Dataset
The dataset contains customer transaction details such as:
- Customer ID
- Purchase Date
- Product Price
- Quantity

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🔍 Project Workflow

### 1. Data Loading
- Import dataset using Pandas

### 2. Data Cleaning
- Remove duplicates
- Handle missing values

### 3. Feature Engineering
- Convert purchase date to datetime
- Create `TotalAmount` feature

### 4. RFM Analysis
- Recency: How recently a customer purchased
- Frequency: Number of purchases
- Monetary: Total spending

### 5. Data Scaling
- Standardize features using `StandardScaler`

### 6. Clustering
- Apply K-Means clustering
- Segment customers into 4 groups

### 7. Data Visualization
- Monthly sales trend
- Correlation heatmap

### 8. Retention Analysis
- Analyze customer purchase frequency
  
---
## 📊 Key Insights
- Identified different customer segments
- High-value and low-value customers distinguished
- Monthly sales trends visualized
- Customer retention patterns observed
